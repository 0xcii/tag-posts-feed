What’s New?》 原文作者：Observers.com 原文编译：ChinaDeFi 之前已经写过了关于以太坊网络即将进行的重大更新：上海升级，通过这次更新以太坊网络将完全过渡到权益证明共识机制。合并增加了以太坊的中心化，上海升级应该能解决这个问题。 以太坊上海网络更新包括几个以太坊改进提案 (EIP)。在关于这次重大更新的前一篇文章中，我们描述了将包含在上海升级中的 EIP。其计划容纳 6 个 EIP： EIP-3540、EIP-3651、EIP-3670、EIP-3855、EIP-3860、EIP-4895 。我们在之前的文章中详细描述了这些 EIP。 以太坊核心开发者会议 151 在 12 月 8 日召开，会上讨论了即将到来的上海升级。这次会议的摘要由以太坊开发者 Tim Beiko 发布。 新增 3 个 EIP：EIP-4200、EIP-4750 和 EIP-5450 。让我们更详细地看看它们是什么。 EIP-4200 ：EOF - 静态相对跳转 这个 EIP 添加了三个新的 EVM 跳转指令：RJUMP、RJUMPI 和 RJUMPV。为了优化网络和降低成本，它们是必需的。 「一个反复出现的讨论主题是 EVM 只有动态跳转的机制。这些提供了一个非常灵活的体系结构，只有 2 条 (!) 指令。然而，这种灵活性是有代价的：它使代码分析更加复杂，并且 ( 部分地 ) 导致需要 JUMPDEST 标记。……这些指令的主要好处是降低了 gas 成本 ( 在部署和执行时 ) 和提供更好的分析性能。」 EIP-4750 ： EOF - 函数 这个 EIP 与 EIP-4200 一样，旨在优化代码。优化是通过将代码划分为几个部分来实现的。 「引入在 EOF 格式 (EIP-3540) 字节码中包含多个代码段的能力，每个代码段代表一个单独的子程序 / 函数。引入了 CALLF 和 RETF 两个新的操作码来调用和返回这样的函数。此外，还引入了 JUMPF 指令来执行跳转到函数的操作。动态跳转指令是不允许的。」 EIP-5450 ：EOF - 堆栈验证 这个 EIP 引入了扩展代码验证。并且旨在优化网络。 「目前，现有的 EVM 实现对每条执行的指令进行大量的有效性检查，例如检查堆栈溢出 / 下溢，是否有足够的 gas 等。此更改旨在通过在部署时进行验证来最大程度地减少运行时所需的此类检查次数。」 在以太坊核心开发者会议 151 上，还讨论了另外两个 EIP 的添加：EIP-663 和 EIP-6064 。EIP-663 的无限 SWAP 和 DUP 指令引入了两个新指令：SWAPN 和 DUPN，它们与 SWAP 和 DUP 的区别在于堆栈深度从 16 个元素增加到 256 个元素。EIP-6046 ：DEACTIVATE 替换 SELFDESTRUCT 引入了对 SELFDESTRUCT 指令的更改，「不删除所有存储密钥，并在帐户随机数中使用特殊值来指示已停用的帐户。」指令 SELFDESTRUCT 也被重命名为 DEACTIVATE。 在以太坊核心开发者会议 151 中，最有趣的信息是上海升级的最后期限。开发人员称 2023 年 3 月是上海硬分叉的最后期限。

    为什么需要代币化？
    对区块链代币化的典型反对理由是：为什么需要代币化？ 答案是效率。系统的效率。区块链代币化使实体经济的效率发生了巨大变化。 通过消除由系统和市场产生的障碍和摩擦，我们将体验一个更加高效的经济，且基本无所谓的“中间人”（第三方）、“守门人”（中心权利方）和创新过程中的障碍。 基于区块链的代币化进程将基于两个基本支柱以此实现系统效率的范式转变：无限集成和无摩擦市场。 代币化将克服集成的技术限制 以银行为例：底层系统是为银行服务而构建的数据库。它是基于许可并构建作为护城河。用户只能通过帐户登陆进行访问。一家大型银行可能与 180 个系统交互，然而在它每年的影响力中，仅仅只能集成 3500 万美元的大型应用程序或服务，并不能集成较小的服务或应用程序。服务不能视为可组合的对象，银行的系统是不可组合的，它是有限的、缓慢的且封闭的。 代币是可以承载无限的逻辑和功能的载体。它们可以无限组合。权利、服务和应用程序可以通过代币无摩擦的集成到系统中。代币化将提供一个更加高效、可访问的基础银行系统。 一个基于区块链的代币化系统，实现了基于 MOAT 银行系统所无法实现的信任、互操作性、隐私和安全。 代币化消除市场创新的摩擦阻碍 今天，我们为功能而设计和建立商业、服务及应用程序超过于开发其潜力。开发人员设计的软件可以按照一套封闭的参数运行，并不向市场开放。在这种设计思想中，可交易的权利是仅排名第二。市场力量作为系统设计的驱动力是不可能的。为什么？因为这个系统已经建立成形了。 范式的转变是在设计系统时将可交易的权利视为第一原则。通过建立围绕可组合和代币化的权利的应用，整个市场的创新和发展无限繁荣。有了代币化权利作为可组合的区块，我们的设计思维就会从 "我需要一个应用程序来启动我的汽车 "转为 "我需要将我的汽车权利代币化，这样我就可以通过第三方租赁获取增量收入。" 我们的设计是为了消除市场中的摩擦，并对市场开放。 代币化将渗透到实体经济的所有领域 为了将这些概念变为现实，这里有6个实体经济进行代币化的尝试，由于区块链代币化带来的系统转变，这些将成为可能。 智能设备。智能餐桌是一种代币化的餐厅餐桌，它知道自己被占用或被预订。代币化减少了摩擦，意味着表格可以实时拍卖或嵌入到其他应用程序（例如 Itinerary Maker 行程表制作者）中。而这种表格的应用场景只是冰山一角。 个人时间。代币化消除了按小时出售时间的摩擦，并使其成为将个人服务有效捆绑到第三方计划和集成产品（例如 Calm App 提供的产品）中的关键。 订阅服务。“激动一年”订阅代币可以集成到各种网站中，因此所有者无需单独付费即可享受内容或服务。这种模式并不新鲜，但由于打破系统边界的成本，在实践中很难实施。 特权代币。“高净值人士”代币远不止代币化的“推荐信”。它可以允许在其之上构建事物。例如，特权代币持有者可以拍卖一个小时的投资顾问推介他们的商品的时间。 服务兑换代币。这与“智能表代币”重叠，但重点是服务而不是物联网设备。例如，酒店客房预订代币允许 1) 通过拍卖发现价值；2）通过包装创造价值；3）通过集成创造价值。 产品所有权代币。这些都是在购买商品时提供的。它使代币的持有者能够随时访问服务，例如维修和更换（例如 iPhone 所有权令牌），并使转售或出租物品的过程变得更加容易。 未来比我们想象的要近得多 我们已经看到无数使用代币化来消除系统和市场中障碍和摩擦的例子。 随着代币化过程变得更加容易，我们将智能代币视为一种更类似于网站或移动应用程序的数字原语，我们将不再问为什么要代币化。 相反，我们会问我如何进行代币化以及何时开始。 作者简介 Weiwu Zhang 是 Smart Token Labs 的创始人兼 CTO。在区块链技术、智能货币设计和交易算法方面拥有超过 9 年的经验。他曾担任 CBA（澳大利亚最大银行）的区块链架构师，担任 R3 全球架构师办公室，并架构了 3 个 altcoins 和 7 个 ICO 项目。 原文链接  ( 23 min )
